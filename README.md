# 🚀 JavaScript Mini Projects

This repository contains a collection of small projects built with **HTML**, **CSS**, and **JavaScript**.

These projects were created to practice core front-end development concepts such as DOM manipulation, event handling, and application logic.

---

## 📚 Projects Included

### 🎯 Guess My Number

A number guessing game where the player tries to guess a random number between 1 and 20.

**Features:**

* Random number generation
* Score system
* Highscore tracking
* Win and lose states

📁 Folder: `Guess My Number`

---

### 🪟 Modal Window

A simple modal component that can be opened and closed using different interactions.

**Features:**

* Open modal with multiple buttons
* Close modal via button, overlay, or Escape key
* Clean UI with overlay effect

📁 Folder: `Modal-Window`

---

### 🎲 Pig Game

A two-player dice game where players compete to reach 100 points.

**Features:**

* Turn-based gameplay
* Dice rolling logic
* Score accumulation
* Winning state with UI feedback

📁 Folder: `Pig Game`

---

## 🛠️ Technologies Used

* **HTML5**

* **JavaScript (Vanilla JS)**

## 🚀 How to Run the Projects

1. Clone this repository:

```bash
git clone https://github.com/your-username/javascript-mini-projects.git
```

2. Open the project folder

3. Navigate to any project folder:

* `Guess My Number`
* `Modal-Window`
* `Pig Game`

4. Open the `index.html` file in your browser
