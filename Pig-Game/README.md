# 🎲 Pig Game

A simple and interactive two-player dice game built with **HTML**, **CSS**, and **JavaScript**.

Players take turns rolling a dice and accumulating points. The first player to reach **100 points** wins the game.

---

## 🚀 Preview

This project is a browser-based game focused on practicing:

- Game logic implementation
- DOM manipulation
- Event handling
- State management
- UI updates based on user interaction

---

## 🕹️ How to Play

1. The game has **2 players**
2. On your turn, click **"Roll Dice"**
3. Each roll adds to your **current score**
4. If you roll a **1**, you lose your current score and turn switches
5. Click **"Hold"** to add your current score to your total score
6. First player to reach **100 points** wins
7. Click **"New Game"** to restart
