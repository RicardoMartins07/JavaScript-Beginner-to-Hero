# 🎮 Guess My Number

A simple and fun number guessing game built with **HTML**, **CSS**, and **JavaScript**.

The player must guess a secret number between **1 and 20**. Each wrong guess decreases the score by 1 point. If the player guesses correctly, the game reveals the number and updates the highscore.

---

## 🚀 Preview

This project is a small browser-based game focused on practicing:

- DOM manipulation
- Event handling
- Conditional logic
- Game state management
- Basic UI styling

---

## 🕹️ How to Play

1. Enter a number between **1 and 20**
2. Click the **Check!** button
3. The game will tell you if the secret number is higher or lower
4. Keep guessing until you find the correct number
5. Your score decreases with each wrong guess
6. Click **Again!** to restart the game
