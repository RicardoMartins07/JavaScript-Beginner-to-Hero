# ⏱️ Stopwatch

A modern and interactive stopwatch built with **HTML**, **CSS**, and **JavaScript**.

This project allows users to start, pause, reset, and track lap times, helping to practice time-based logic and dynamic DOM updates.

---

## 🚀 Preview

This project focuses on building a real-time application using:

- JavaScript timers (`setInterval`)
- DOM manipulation
- State management
- Dynamic UI updates

---

## 🕹️ Features

- ▶️ Start the stopwatch
- ⏸️ Pause and resume functionality
- 🔄 Reset the timer
- 🏁 Record lap times
- 📋 Dynamic lap list display
- 🎯 Clean and responsive UI

---

## 🧠 How It Works

- The stopwatch starts from **00:00**
- Time increases every second
- Users can pause and resume without losing progress
- Each lap records the current time and displays it in a list
- Reset clears both the timer and lap history
