# ⏳ Countdown Timer

A simple and modern countdown timer built with **HTML**, **CSS**, and **JavaScript**.

This project allows users to set a custom time and count down to zero with start, pause, and reset functionality.

---

## 🚀 Preview

This project focuses on building time-based logic using:

- JavaScript timers (`setInterval`)
- DOM manipulation
- User input handling
- Real-time UI updates

---

## 🕹️ Features

- ▶️ Start countdown from custom time
- ⏸️ Pause and resume functionality
- 🔄 Reset the timer
- ⌛ Real-time countdown display
- 🎯 Clean and simple UI

---

## 🧠 How It Works

- The user inputs **minutes and seconds**
- The timer converts the input into total seconds
- Every second, the time decreases
- When the timer reaches **00:00**, it stops and displays a message
