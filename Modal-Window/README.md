# Modal Window

A simple modal window project built with **HTML**, **CSS**, and **JavaScript**.

This project demonstrates how to create, open, and close a modal using different user interactions such as button clicks, overlay clicks, and keyboard events.

---

## 🚀 Preview

This project is a small front-end exercise focused on practicing:

- DOM manipulation
- Event handling
- Class toggling
- Keyboard interaction
- Basic UI design

---

## ✨ Features

- Open the modal by clicking any of the three buttons
- Close the modal by clicking the close button
- Close the modal by clicking on the overlay
- Close the modal by pressing the **Escape** key
- Clean and simple user interface

---

## 🛠️ Technologies Used

- **JavaScript (Vanilla JS)**
